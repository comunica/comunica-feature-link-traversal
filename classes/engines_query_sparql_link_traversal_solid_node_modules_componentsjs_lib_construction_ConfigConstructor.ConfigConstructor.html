Class ConfigConstructor<Instance>

Creates instances of raw configs using the configured creation strategy.

A raw config MUST adhere to the following shape:

  • requireName: required
  • requireElement: optional
  • arguments: optional

Arguments will recursively be converted to instances using IArgumentConstructorHandler's.

This will always create unique instances of configs. If you want to make sure that instances are reused, be sure to call ConfigConstructorPool instead.
