Class ConfigConstructorPool<Instance>

Manages and creates instances of components based on a given config.

This accepts different config variants, as supported by the configured IConfigPreprocessor's.

This will make sure that configs with the same id will only be instantiated once, and multiple references to configs will always reuse the same instance.
