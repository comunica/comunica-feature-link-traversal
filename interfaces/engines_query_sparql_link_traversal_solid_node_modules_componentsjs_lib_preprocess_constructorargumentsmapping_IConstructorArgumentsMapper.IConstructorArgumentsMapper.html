    Map config with param instantiations to a raw config according to the given constructor arguments definition.

    For example, the constructor arg { key: '"param0"', value: 'http://example.org/param0' } with config { 'http://example.org/param0': '"abc"' } will be mapped to { key: '"param0"', value: '"abc"' }.

    Parameters

    • configRoot: Resource

      The root config resource that we are working in.

    • constructorArgs: Resource

      Object mapping definition inside the constructor arguments.

    • configElement: Resource

      Part of the config resource to look for parameter instantiations as predicates.

    • genericsContext: GenericsContext

      Context for generic types.
