# Comunica Bus Factory Aggregated Store

[![npm version](https://badge.fury.io/js/%40comunica%2Fbus-factory-aggregated-store.svg)](https://www.npmjs.com/package/@comunica/bus-factory-aggregated-store)

A bus for constructing aggregated stores.

This module is part of the [Comunica framework](https://github.com/comunica/comunica),
and should only be used by [developers that want to build their own query engine](https://comunica.dev/docs/modify/).

[Click here if you just want to query with Comunica](https://comunica.dev/docs/query/).

## Install

```bash
$ yarn add @comunica/bus-factory-aggregated-store
```

## Bus usage

* **Context**: `"https://linkedsoftwaredependencies.org/bundles/npm/@comunica/bus-factory-aggregated-store/^1.0.0/components/context.jsonld"`
* **Bus name**: `ActorAggregatedStoreFactory:_default_bus`
